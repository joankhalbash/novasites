# Novasites

Dette er en React/Vite hjemmeside klar til Vercel.

## Kør lokalt

```bash
npm install
npm run dev
```

## Deploy til Vercel

1. Upload projektet til GitHub, eller importer mappen via Vercel CLI.
2. Opret et nyt projekt i Vercel.
3. Vælg framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Tilføj domænet `novasites.dk` i Vercel.
7. Sæt DNS-records hos Simply efter Vercels instruktioner.

Vigtigt: Slet ikke MX-records hos Simply, fordi `kontakt@novasites.dk` skal fortsætte med at virke.
